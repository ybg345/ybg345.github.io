Ophelos
Python, Django, HTML, CSS, Bootstrap, MySQL, JavaScript etc.

Professional Project		 This projects is designed to maintain all the technical and non-technical information of our IIG (International Internet Gateway) downstream partner. It's currently live and around 50 peoples are using it in daily basis. Features includes but not limited to -

✫ Animated text and design at Welcome Page ✫ Sign up and Login Functionality ✫ Forms for inserting data into database ✫ Update & Delete inserted information ✫ Export data as Excel file ✫ Filter information using search as we type based on any column ✫ Activity log of user activity e.g. username & time of creation/update of a record.
